## ITOSi MicroServices
微服务框架实践，基于[ Spring Cloud ](http://projects.spring.io/spring-cloud/)框架
## 功能
框架分为四个层面提供服务

* 接口服务（API）
* 汇聚服务（Composite）
* 核心服务（Core）
* 支持服务（Support）


## 文档
#### 系统需求
1. JDK1.8+
2. 内存4G＋，大吞吐量的数据处理需要更大的内存
3. 磁盘100G＋，文件型Channel会需要更大的磁盘空间
4. 目录权限，安装目录要具备读／写／执行权限
